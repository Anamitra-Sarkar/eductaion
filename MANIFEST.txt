HOW TO USE
================================================================================

1. LOCAL DEVELOPMENT
   cd backend
   pip install -r requirements.txt
   python seed.py
   uvicorn main:app --reload
   
   Open attendx.html in browser
   Login: admin@college.edu / Admin@123

2. PRODUCTION DEPLOYMENT
   - Follow steps in DEPLOYMENT.md
   - Push backend to GitHub
   - Deploy to Render.com
   - Update config.js with Render API URL
   - Deploy frontend to Vercel

3. DATABASE
   - SQLite for local development (auto-created)
   - PostgreSQL for production (set DATABASE_URL in .env)
   - All migrations handled by SQLAlchemy on startup

4. API DOCUMENTATION
   http://localhost:8000/docs (Swagger UI)
   http://localhost:8000/redoc (ReDoc)

SUPPORT & TROUBLESHOOTING
================================================================================

Issue: "Connection refused" error
Solution: Check if backend is running at http://localhost:8000

Issue: Authentication fails
Solution: Verify SECRET_KEY in .env, check token expiry

Issue: Database errors
Solution: Run `python seed.py` to initialize tables

Issue: Frontend not loading
Solution: Check browser console, verify attendx.html is accessible

Issue: Data not showing
Solution: Ensure seed.py completed, check API logs

Full troubleshooting guide in DEPLOYMENT.md
